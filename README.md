# 🚀 NOGUEIRA - Web Developer Portfolio

Bem-vindo ao repositório do meu portfólio profissional. Este projeto foi desenvolvido para apresentar meus trabalhos mais recentes, minhas habilidades técnicas e minha paixão por criar experiências web excepcionais.

![Aesthetic Preview](https://raw.githubusercontent.com/Santozs2/Portfolio-Nogueira/main/src/assets/preview.png) *(Nota: Você pode adicionar uma screenshot do seu site aqui após o deploy)*

## ✨ Overview

Este portfólio é um reflexo do meu compromisso com o design premium e a engenharia de precisão. Focando em animações fluidas, tipografia marcante e uma experiência de usuário (UX) impecável, ele utiliza tecnologias modernas para garantir alta performance e plena responsividade.

### Principais Funcionalidades:
- **Design Minimalista & Premium**: Estética de alto impacto com foco em tipografia e alto contraste.
- **Micro-interações**: Animações fluidas utilizando Framer Motion para uma sensação tátil e moderna.
- **Dark Mode Nativo**: Interface totalmente adaptável para temas claro e escuro.
- **Modal de Projetos Dinâmico**: Visualização detalhada de cada projeto com carrossel de imagens integrado.
- **Cursor Customizado**: Sistema de cursor interativo que reage a elementos da interface.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Ícones**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Como Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/Santozs2/Portfolio-Nogueira.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
