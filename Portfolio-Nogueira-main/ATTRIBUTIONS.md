# Atribuições e Créditos

Este projeto foi desenvolvido com o auxílio de diversas tecnologias e recursos de código aberto. Abaixo estão as atribuições para as ferramentas e bibliotecas utilizadas:

## Tecnologias Core
- **React**: Biblioteca principal para construção da interface.
- **TypeScript**: Superset do JavaScript para tipagem estática.
- **Vite**: Build tool e servidor de desenvolvimento de alta performance.
- **Tailwind CSS**: Framework de estilização utilitária.

## Bibliotecas de Terceiros
- **shadcn/ui**: Componentes de interface baseados em Radix UI e Tailwind CSS. [MIT License](https://github.com/shadcn-ui/ui/blob/main/LICENSE.md).
- **Lucide React**: Biblioteca de ícones vetoriais. [ISC License](https://github.com/lucide-software/lucide/blob/main/LICENSE).
- **Motion (Framer Motion)**: Biblioteca de animações para React. [MIT License](https://github.com/framer/motion/blob/main/LICENSE.md).
- **React Router**: Gerenciamento de rotas e navegação SPA. [MIT License](https://github.com/remix-run/react-router/blob/main/LICENSE.md).
- **Sonner**: Biblioteca de notificações (toast). [MIT License](https://github.com/emilkowalski/sonner/blob/main/LICENSE).

## Recursos Visuais
- **Skill Icons**: Ícones de tecnologias fornecidos por [skillicons.dev](https://skillicons.dev/).
